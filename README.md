## MCP Server Configuration Guide

### Basic Setup
1. Install Node.js (v16 or later)
2. Install the MCP Server package: `npm install -g @modelcontextprotocol/server`
3. Create a configuration file (mcp_config.json)

### Video Reference

### Common MCP Commands
```bash
# Start MCP server
mcp-server start

# List available tools
mcp-server tools
```
